# SOC Incident Response Investigation

## Objective
Investigate a simulated security alert and determine if it is a true positive.

## Tools Used
- SIEM
- Windows Event Logs
- Network Logs

## Investigation Steps
1. Reviewed alert details
2. Analyzed source IP and timestamps
3. Correlated logs across systems
4. Identified malicious activity

## Findings
- Detected unauthorized login attempts
- Identified suspicious IP addresses

## Response Actions
- Recommended IP blocking
- Password reset
- Monitoring for future activity

## Lessons Learned
This project improved my understanding of incident response workflows and alert triage.
